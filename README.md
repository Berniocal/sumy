# Šumy (PWA)

Instalovatelná PWA aplikace generující šumy (white/pink/brown) a presety (vodopád, vítr, ventilátor, vysavač)
pomocí Web Audio API (bez audio souborů). Offline díky Service Workeru.

## Spuštění lokálně
Je potřeba spustit přes lokální server (kvůli AudioWorklet + SW).

- VS Code: Live Server
- Node: `npx serve`

Pak otevři `index.html` přes server a klikni **Spustit**.

## Nasazení
Nahraj složku na GitHub Pages (root) a otevři URL. V prohlížeči nabídne instalaci.
